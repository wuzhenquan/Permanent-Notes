## 录入原则

- 加入的内容必须是自己加工和处理过的。
- Fleeting Notes 不在这进行处理。
- [Make sure that it is expressed clearly without needing a lot of context](https://youtu.be/GpV47rUYk8I?t=251)
- 以后每更新一个知识点，就发布一次 Release。
